# Review Pull Request Comment

引数 `$ARGUMENTS` で指定された PR(省略時はカレントブランチに紐づく PR)のコメントを `gh` で取得し、内容を評価する。

## 目的

PR に付いたコメント(通常コメント・レビューコメント)の内容を検討し、対応方針の判断材料を提供する。

## 引数仕様(`$ARGUMENTS`)

- PR 番号(省略可): 評価対象の PR を指定する。省略時はカレントブランチに紐づく PR を対象とする。

## 実行手順

1. `$ARGUMENTS` に PR 番号があればそれを対象とする。無ければ `gh pr view` でカレントブランチに紐づく PR を特定する。
2. `gh pr view` または `gh api`(レビュースレッド等、`gh pr view` で取得できない情報が必要な場合)で PR のコメントを取得する。
3. 必要に応じてコメントが指す該当箇所を Read/Grep で確認する。
4. コメントごとに内容を評価する(指摘の妥当性、対応の必要性、優先度など)。

## 出力形式

```
## PR #<番号> のコメント評価

| コメント内容 | 該当箇所 | 評価 | 対応の必要性 |
| --- | --- | --- | --- |
```

## 動作上の注意

- コメントの内容を推測で補わない。取得した情報のみを根拠にする。
