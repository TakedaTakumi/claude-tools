# Security

このリポジトリは個人ツールとして公開しています。サポート・修正対応は best-effort で、保証はありません（[LICENSE](LICENSE) 参照）。

## 脆弱性を見つけたら

`install.sh` / `bootstrap.sh` の挙動・Claude Code が読み込む Markdown コンテンツ（`commands/` / `agents/` / `skills/` / `config/`）に脆弱性を見つけた場合は、可能であれば **GitHub Security Advisories の "Report a vulnerability"** を使ってください。

- リポジトリ: <https://github.com/TakedaTakumi/claude-tools>
- 報告フォーム: リポジトリの **Security** タブ → **Advisories** → **Report a vulnerability**

公開 Issue ではなく Advisories を使うことで、修正前に攻撃手段が広まるのを避けられます。

対応は時間が取れる範囲で行います。Critical / High に該当しそうな問題は優先しますが、確約はできません。
